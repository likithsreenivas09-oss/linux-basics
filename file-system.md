# Linux File System

## What is Linux?
Linux is an operating system that uses a hierarchical file system.

## Important Directories
- / – root directory
- /home – user files
- /etc – configuration files
- /var – log files
- /bin – system commands

## Commands I Used
- ls
- cd
- pwd

## What I Learned
I learned how Linux organizes files and how to move between directories using the terminal.
